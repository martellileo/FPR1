# FPR1 - Ferramentas de Programação 1

Repositório responsável por armazenar meus projetos e atividades desenvolvidos durante a disciplina de Ferramentas de Programação 1 (FPR1).

## Sobre a Disciplina

Este repositório contém projetos e exercícios focados em tecnologias de desenvolvimento web com Java, incluindo JavaServer Faces (JSF) e Java Persistence API (JPA).

## Estrutura do Repositório

O conteúdo está organizado nas seguintes pastas:

* **/Atividade Avaliativa - 1**: Código-fonte da primeira atividade avaliativa.
* **/Atividade Avaliativa - 2**: Código-fonte da segunda atividade avaliativa.
* **/facelets**: Projeto utilizando a tecnologia de templates Facelets.
* **/introducao-jsf**: Projeto introdutório sobre JavaServer Faces (JSF).
* **/jpa-introducao**: Projeto introdutório sobre a Java Persistence API (JPA) para gerenciamento de banco de dados.
* **/libraries**: Contém as bibliotecas e dependências (.jar) necessárias para os projetos.

## Tecnologias Utilizadas

As principais linguagens e frameworks utilizadas nos projetos são:

* **Java**
* **JavaServer Faces (JSF)**
* **Java Persistence API (JPA)**
* **Facelets**
* **HTML**
* **CSS / SCSS**
